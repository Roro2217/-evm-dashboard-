# EVM Project Dashboard — Interactive Demo

An interactive Earned Value Management (EVM) dashboard built as a single-page HTML application. Demonstrates real-time project performance monitoring with planned value, earned value, and actual cost tracking.

## Live Demo

👉 **[View the live dashboard](https://<your-username>.github.io/evm-dashboard/)**

## Features

- **Interactive timeline slider** — scrub through 12 months of project data
- **4 key EVM metrics** — SV, CV, SPI, CPI with color-coded status
- **Cumulative S-curves** — PV, EV, AC plotted against BAC baseline
- **Variance bar chart** — monthly schedule and cost variance breakdown
- **Auto-status badge** — ON TRACK / AT RISK / OVER BUDGET based on CPI & SPI thresholds
- **Estimate at Completion (EAC)** — real-time forecast based on current CPI

## Quick Deploy to GitHub Pages

1. Create a new repo named `evm-dashboard` on GitHub
2. Upload `index.html` (or push this folder)
3. Go to **Settings → Pages → Source → Deploy from branch (main)**
4. Your dashboard is live at `https://<your-username>.github.io/evm-dashboard/`

## Technology

- Pure HTML/CSS/JS — zero build step, single file
- [Chart.js 4.4.1](https://www.chartjs.org/) for data visualization
- [DM Sans](https://fonts.google.com/specimen/DM+Sans) + [JetBrains Mono](https://www.jetbrains.com/lp/mono/) typography
- Dark theme optimized for visual impact
